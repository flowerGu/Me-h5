一.按钮

使用说明：按钮元件在unit.less文件，目前共两类，前缀分别为标准按钮u_btn及表单按钮u_btn_form,使用方式例如：

<span class="u_btn_default">出借</span>

<span class="u_btn_disabled">售罄</span>

<a class="u_btn_form_default">登录</a>

<a class="u_btn_form_disabled">注册</a>

<input type="submit" class="u_btn_bottom" value="下一步">

扩展说明：新增按钮需新增按钮类别基础样式，例如新增文本类型按钮,需要新增u_btn_text基础样式,同类按钮不同状态下需引入基础样式，增加特殊样式,例如u_btn_text_primary,u_btn_text_disabled等...

出借 售磬

2.表单按钮

登录 注册

二.标签

使用说明：标签元件在unit.less文件使用方式例如：

<span class="u_btn_tag">新手标</span>

<span class="u_btn_tag_disabled">热门推荐</span>

<span class="u_btn_tag_dialog">热门推荐</span>

扩展说明：新增标签需在标签类别下新增样式，命名以.u_btn_tag开始，后缀需具有描述性。

四.表单

使用说明：输入框元件在unit.less文件使用方式例如：


<ul>
    <li class="u_form_single">
        <input type="text" placeholder="6-16位数字、字母、符号组合" class="u_input_full">
    </li>
  </ul>


其中.u_input_full为普通输入框，.u_input_per为带label输入框（增加class为u_form_send的button，可以增加发送短信验证码按钮

扩展说明：新增表单元素需在表单类别下新增样式，父元素命名以.u_form开始，输入框元素以u_input开始后缀需具有描述性。
